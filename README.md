# DS4002-CS3
This repository contains the topic overview, deliverable, data, rubric, and reference materials necessary to complete the Case Study.

## Data
The necessary data is located in the "DATA" folder and includes both a cleaned dataset to be used for your model as well as the original and complete World Happiness Report data.

## Reference Materials
The `materials/` folder contains helpful resources related to the context of the social issue, tips for time-series regression analysis and developing a predictive model, as well as the complete dataset's source website.
